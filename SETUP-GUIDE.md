## ðŸ†˜ Troubleshooting

### Problem: "404 - Page not found"
**Solution**:
- Wait 2-3 minutes after enabling GitHub Pages
- Check that files are in the root of the repository (not in a subfolder)
- Verify GitHub Pages is enabled in Settings

### Problem: "File not found" when clicking tools
**Solution**:
- Make sure the `tools` folder is uploaded correctly
- Check that `bank-to-xero.html` is inside the `tools` folder
- File names are case-sensitive

### Problem: Converter doesn't work
**Solution**:
- Make sure you're uploading a CSV file
- Check that your browser allows downloads
- Try a different browser (Chrome recommended)

### Problem: Can't upload files to GitHub
**Solution**:
- Try a different browser
- Check your file size (should be under 25MB total)
- Upload files one at a time if batch upload fails
